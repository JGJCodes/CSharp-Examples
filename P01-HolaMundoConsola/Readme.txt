Proyecto de Visual Studio 
Aplicacion de consola en C#
Autor: Jorge Guzman Juarez (JGJCodes)

++++++++Descripcion+++++++++++
Crear una aplicacion que imprima el mensaje 
"Hola Mundo" en la consola del sistema.
La aplicacion debe ser creada por bloc de notas 
y utlizando la terminal del sistema.

Comando de compilacion:
c:\Windows\Microsoft.Net\Framework\v4.0\csc HolaMundo.csc

Comando de ejecucion:
c:\User\Win\Document\Holamundo\HolaMundo.exe
