++++++++Descripcion+++++++++++
Es necesario crear un libro electronico de valoraciones para
pode almacenar y poder hacer consultas sobre las valoraciones
recibidas.
Tambien se debe poder hacer calculos y promedios entre todas 
las valoraciones recibidad.
Las valoracciones seran en numeros decimales=float, que pueden
ir de 0 a 5, debemos poder listar las valoraciones desde mas bajas,
desde mas altas y la nota promedio entre ellas.
